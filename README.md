# Image Captioning Project (CNN + LSTM)

## Overview
This project generates captions for input images using:

- CNN Encoder (InceptionV3 features)
- LSTM Decoder
- Word Embeddings
- Greedy Search decoding
- BLEU evaluation metrics

This repository includes:
- Scripts for preprocessing images
- Caption cleaning + tokenization
- Feature extraction
- Sequence generation
- Model building
- Model training
- BLEU evaluation
